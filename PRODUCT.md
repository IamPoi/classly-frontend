# Classly 프로덕트 정리

## 한 줄 정의
소규모 학원 원장·선생님을 위한 **출결 자동화 + 학부모 소통 자동화** 웹 서비스

---

## 타겟
- **1차**: 소규모 학원 원장 / 선생님
- **2차 (확장)**: 초등학교 선생님 개인

---

## 핵심 기능 2축

### 1. QR 출결 자동화
- 교실 문/책상에 QR 스티커 1회 부착
- 학생이 입장 시 QR 스캔 → 브라우저 열림 (앱 설치 없음)
- GPS 위치 확인 (학원 반경 100m 이내 검증)
- 수업 시간 윈도우 검증 (수업 시작 5분 전 ~ 10분 후)
- 출결 처리 → 학부모 카톡 알림 자동 발송

**대리출결 방지**
- GPS: 학원 근처에서만 출결 가능
- 시간 윈도우: 수업 시간 외 스캔 불가

### 2. AI 학부모 메시지 생성
- 선생님이 학생 성취도 메모 입력
- AI가 학부모 상담 메시지 초안 자동 생성
- 카카오톡으로 바로 발송

---

## 사용자별 역할

| 역할 | 주요 기능 |
|------|---------|
| 원장 | 회원가입, 학원 정보 설정, 선생님/반 관리 |
| 선생님 | 학생 관리, AI 메시지 생성, 출결 현황 확인 |
| 학생 | QR 스캔으로 출결 (웹, 앱 없음) |
| 학부모 | 카톡 알림 수신만 (앱 없음) |

---

## 회원가입 플로우 (원장)
```
원장 회원가입
→ 학원 이름 + 주소 입력
→ 카카오 주소 API로 위경도(lat, lng) 변환
→ DB 저장
→ QR 코드 자동 생성 (학원 ID + 반 ID 포함)
→ QR 스티커 출력
```

## 출결 플로우 (학생)
```
QR 스캔
→ Classly 출결 전용 웹페이지 열림
→ GPS 위치 권한 요청 (브라우저 기본 기능)
→ 학원 반경 + 수업 시간 검증
→ 출결 처리
→ 학부모 카톡 자동 발송 "OO이 출석했습니다 ✓"
```

---

## 기술 스택
- **프론트**: Next.js + Tailwind
- **주소 → 좌표**: 카카오 주소 API (Geocoding)
- **GPS 검증**: 브라우저 `navigator.geolocation`
- **학부모 알림**: 카카오 알림톡

---

## 비즈니스 모델
- 3개월 무료 → 레퍼런스 확보
- 이후 학생 수 기반 월 구독
  - 소규모 (~ 30명): 월 3만원
  - 중규모 (~ 60명): 월 6만원
  - 대규모 (60명+): 월 10만원
- 인터뷰 참고: 월 12만원까지 수용 가능하다는 반응 있었음

---

## 경쟁 현황
- 현재 학원에서 쓰는 유료 툴: 매쓰홀릭, 클래스카드
- 출결·학부모 소통을 한 번에 해결하는 툴은 없음

## 인터뷰 인사이트 (2026.03.30)
- 진도 기록은 학생 책에 아날로그로 함 → 디지털화 기회
- 학부모 연락은 카톡/전화/문자 혼용 → 카톡 단일화 기회
- 월간 리포트는 "귀찮아서" 안 보냄 → AI 자동화 니즈 명확
- 출결 자동화가 가장 강한 페인포인트로 확인됨
- 지불 의향 있음 (기존 유료 툴 사용 중)
